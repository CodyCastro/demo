Модуль 1

1) Откройте терминал и введите это:

sudo -i

пароль student

2) Создаём папку для приложения, а также для бд и даём права на неё

на всякий случай, перед созданием удалим:

rm -fr /db /home/student/DemoApp

теперь создаём:

mkdir /home/student/DemoApp
mkdir /db
chown reddatabase /db

3) Клонируем репозитории

1 смена:

git clone https://github.com/igorten21/DemoApp.git /home/student/DemoApp
git clone https://github.com/igorten21/db1.git /db

2 смена:

git clone https://github.com/igorten21/DemoApp.git /home/student/DemoApp
gitclone https://github.com/igorten21/db2.git /db

4) Открываем RedExpert для проверки

Для проверки нужно:

  1. создать подключение - кнопка New Connection

  2. Ввести указанные настройки

Connection Name: demo

Database File: /db/demo.fdb

User Name: sysdba

Password: 12345678 (либо student)

ЗАТЕМ НАЖАТЬ Save И ЗАТЕМ Connect

5) Смотрим таблицы и т.д (делайте умный вид, бд уже вся готовая)

6) Заходим в VSCode, в левом верхнем углу жмём File, затем Open Folder, затем ищем папку DemoApp и снизу кнопка открыть

7) Прописываем в VSCode терминале

dotnet run

Всё должно работать, проверяйте

8) Делайте умный вид или исправляйте мои ошибки
